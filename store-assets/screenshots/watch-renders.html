📸 How to capture these

  1. Open this file in a browser (Chrome recommended)
  2. Right-click any watch → Inspect → Device toolbar → set to a fixed size
  3. Or just screenshot each watch frame individually
  4. Use ⌘+Shift+4 on Mac to select just the watch area
  5. The dark background will crop cleanly for your website

💡 Tip: For App Store screenshots, Apple requires 410×502 px for Series 10 (46mm). You can resize these captures to match.
